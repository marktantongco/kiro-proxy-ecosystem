4

❓ Ecosystem FAQ & Incident Management

What is the difference between kiro-cli and kiro-gateway? ▼
kiro-cli: The native binary used to perform OIDC Builder ID logins and update credential stores.
kiro-gateway: The python proxy server running in the background to translate active OpenAI/Anthropic calls dynamically. You need both to retain continuous communication.
How do I triage a 403 Forbidden / 502 Bad Gateway monthly quota issue? ▼
This occurs when the active Builder ID monthly free quota has been fully consumed. The system automatically degrades to its 13 cached fallback models. To recover full operations, register a new free Builder ID, run kiro-cli login, and restart the gateway service: systemctl --user restart kiro-gateway.service.
What common operational mistakes must be avoided? ▼
1. Duplicate Port Bindings (Errno 98): Running manual python sessions while the systemd unit is active.
2. Dead Lowecase Variables: Leaving dead variables (like port 2080) in the environment.
3. NO_PROXY Pollution: Adding AWS backend endpoints inside the NO_PROXY string, which bypasses geo-routing entirely.
